              [--device DEVICE_ID] [--sound] [--message MESSAGE]

Find My iPhone CommandLine Tool

options:
  -h, --help            show this help message and exit
  --username USERNAME   Apple ID to Use
  --password PASSWORD   Apple ID Password to Use; if unspecified, password
                        will be fetched from the system keyring.
  -n, --non-interactive
                        Disable interactive prompts.
  --delete-from-keyring
                        Delete stored password in system keyring for this
                        username.
  --list                Short Listings for Device(s) associated with account
  --llist               Detailed Listings for Device(s) associated with
                        account
  --locate              Retrieve Location for the iDevice (non-exclusive).
  --device DEVICE_ID    Only effect this device
  --sound               Play a sound on the device
  --message MESSAGE     Optional Text Message to display with a sound
  --silentmessage SILENTMESSAGE
                        Optional Text Message to display with no sounds
  --lostmode            Enable Lost mode for the device
  --lostphone LOST_PHONE
                        Phone Number allowed to call when lost mode is enabled
  --lostpassword LOST_PASSWORD
                        Forcibly active this passcode on the idevice
  --lostmessage LOST_MESSAGE
                        Forcibly display this message when activating lost
                        mode.
  --outputfile          Save device data to a file in the current directory.
  --session-directory SESSION_DIRECTORY
                        Path to save session information
  --region REGION       Server region - global or china
